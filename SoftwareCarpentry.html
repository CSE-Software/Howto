Overview

This is the tutorial for who want to contribute for Software Carpentry website. This tutorial only contain basic instructions with some specific steps. Other questions or issues contact Duong Do

Prerequisites

Basic understanding on Jekyll.

Steps

  1. Create normal HTML files, but do not include "html", "title", "head", and "body" tag
  2. Type these following at the beginning of the file:
    ---
    layout: page-fullwidth
    permalink: /Name of HTML file (without .html)/
    title: Your title
    ---
  3. Put your files into the pages folder
  4. Come to data/navigation.yml and add the correspond links for the files
