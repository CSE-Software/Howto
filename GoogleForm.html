Idea: Using Google API Console to collect information from Google Form

Steps:

  1. Using Google API Console to generate .json file with private key and ID
  2. Research on Google Spreadsheets Python API
  3. Open the .json file to see all the necessary information. Share Google spread sheet with client email in the .json file.
  4. Write a script to process the data collected.
    There are 3 necessary parts on the script.
  1. Get credentials from Google by open .json file with open('google_key.json') as f: config = json.load(f) creds = SignedJwtAssertionCredentials( config['client_email'], config['private_key'], ['https://spreadsheets.google.com/feeds'] ) return gspread.authorize(creds)
  2. Get the information from Google Form and process it gc = get_google_client() doc = gc.open_by_key({{ fill in the Google Sheet ID }}) sheet = doc.worksheet({{ fill in the Google Sheet Tab }}) row_count = sheet.row_count for row in range(2, row_count + 1): values = sheet.row_values(row) if not values[0]: break process_value(values)
